# DHO800_BatteryPack

**为DHO800/900系列示波器设计的电池包（BatteryPack Designed for DHO800/900 Series Oscilloscopes）**

**项目简介**：采用两节21700电池供电，可更换电池设计，便于拆卸通过安检。续航时间1小时。不影响原设备的任何功能(散热、壁挂、接口插拔)。支持电量显示。支持对外PD供电输出。

**已知问题**：更换电池后需插电激活使用。
